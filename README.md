# Paza Sauti La Haki - Élève la Voix de la Justice

## À propos
**Paza Sauti La Haki** (Élève la Voix de la Justice) est une initiative portée par des jeunes étudiants de différentes universités de Goma. L'objectif principal est de rendre la justice accessible à tous en République Démocratique du Congo (RDC) et de promouvoir la paix ainsi que la gestion pacifique des conflits.

## Fonctionnalités
- **Design Moderne et Responsive :** Interface utilisateur soignée avec une charte graphique aux couleurs de la RDC et de la justice (Bleu, Rouge, Jaune).
- **Animations Fluides :** Utilisation de `Framer Motion` pour une expérience utilisateur dynamique (apparitions, transitions).
- **Section Blog Interactive :** Articles de sensibilisation sur la justice et la paix avec affichage détaillé en modal.
- **Présentation de l'Équipe :** Liste des membres avec une vue modale pour les détails et une page complète pour voir toute l'équipe.
- **Navigation Intuitive :** Menu fixe avec défilement fluide (smooth scrolling) vers les différentes sections.

## Technologies Utilisées
- **React 19** : Bibliothèque JavaScript pour l'interface utilisateur.
- **TypeScript** : Pour un code typé et sécurisé.
- **Tailwind CSS** : Framework CSS pour le stylisme rapide et responsive.
- **Framer Motion** : Pour les animations complexes.
- **Lucide React** : Pour les icônes vectorielles modernes.

## Structure du Projet
- `/components` : Contient les composants React modulaires (Navbar, Hero, Mission, Team, Blog, Footer).
- `App.tsx` : Composant racine assemblant les sections.
- `types.ts` : Définitions des types TypeScript (TeamMember, BlogPost, etc.).
- `index.html` : Point d'entrée HTML avec la configuration Tailwind et les polices Google Fonts.
